# DOM Snapshots

Directory for storing Antigravity UI DOM snapshots for analysis and debugging.

## Structure
- `main-window/` - Main Antigravity workbench DOM
- `agent-manager/` - Agent Manager panel DOM
- `cascade-panel/` - Cascade panel (chat) iframe DOM

## Naming Convention
`YYYY-MM-DD_HH-MM_<component>_<description>.html`
